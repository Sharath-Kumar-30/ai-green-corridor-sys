AI Based Green Corridor System for Ambulances - Demo

Project Overview

This project aims to create an AI-enabled healthcare service that facilitates the rapid movement of ambulances through traffic using a green corridor system. The system includes various screens for users, ambulance teams, traffic control, and hospitals.

Features

User Screen

Users can:

  • Request an ambulance
  • Share their location
  • Upload pictures and PDFs

Ambulance/Emergency Response Teams Screen

Teams can:

  • Receive notifications of ambulance requests
  • Acknowledge requests and send notifications back to users
  • Notify traffic control teams

Traffic Control Screen

Traffic control can:

  • Receive notifications and vague paths for ambulances
  • Upload videos for fake video detection
  • Receive green signals for traffic junctions

Hospital Screen

Hospitals can:

  • Receive notifications about incoming patients
  • View patient symptoms and prepare in advance
  • Access uploaded PDFs and images

Architecture and Flow of Services

The system is built using a microservices architecture with the following components:

  • Frontend: Angular application for user interaction.
  • Backend: FastAPI application handling business logic and API endpoints.
  • Database: PostgreSQL for data storage.
  • Message Broker: RabbitMQ for communication between services.
  • Object Storage: MinIO for storing uploaded files.
  • Task Queue: Celery for background processing.
  • Video Detection: YOLOv5 for detecting ambulance traffic scenarios.

Endpoints

The following are the main API endpoints:

  • Users:
    POST /api/v1/users/request-ambulance
  • Ambulance:
    GET /api/v1/ambulance/notifications
  • Traffic Control:
    POST /api/v1/traffic/notify
  • Hospital:
    GET /api/v1/hospital/notifications

How to Demo

To demonstrate the system:

  1. Start the Docker containers using the provided docker-compose.yml file.
  2. Access the frontend application in a web browser.
  3. Simulate a user requesting an ambulance.
  4. Show the ambulance team's response and traffic control notifications.
  5. Demonstrate the hospital's preparation for incoming patients.
  6. Upload videos for traffic detection and show the fake detection results.
